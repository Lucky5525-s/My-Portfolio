# Personal Portfolio Website

A modern and responsive personal portfolio website built using HTML and CSS. This project showcases my profile, interests, social media links, and professional aspirations in a clean and visually appealing interface.

## Features

- Responsive design for different screen sizes
- Attractive landing page with profile image
- Animated typing text effect
- About Me section
- Social media integration
- Hire Me button
- Smooth scrolling navigation
- Modern dark-themed UI
- Custom styling with CSS animations

## Technologies Used

- HTML5
- CSS3

## Sections Included

- Home
- About Me
- Skills
- Services
- Contact

## Project Purpose

This portfolio website was created to practice front-end web development and to showcase my personal profile, interests, and growing skills as an engineering student and aspiring web developer.
